This music rotation site was created in January 2006 as a personal website to reccommend songs by my favorite musicians to the rest of the world.

The site layout changed several times between 2006 and 2008. Each layout features the images of a musician. Navigation for all layouts existed as an image-map in the main banner.

The site also utilizes a comment box which also serves as a guestbox for each rotation.

The site was built with HTML and CSS in Notepad. Images for the layout and site were created in Adobe Photoshop 7.0.
