This online portfolio was in January 2012 created as a class assignment for COGS 187A at UCSD. The goal was to create a functional, self-expressive portfolio to show our work.

The site contains a home page, portfolio pages, and a contact page. Enlarged images pop-up in a lightbox (art, photos, and screenshots). Buttons for external sites on the upper-right animate when scrolled-over.

The site was built with HTML, CSS, and Javascript -- with help from Bootstrap and Fancybox. All images on the site were draw or taken by myself. Programs used in the creation of the site include Google Chrome, Notepad++, Adobe Photoshop CS5 and Lightroom 3.
